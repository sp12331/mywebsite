Project 3

Fingerprint Based Attendance System

The Fingerprint Based Attendance System, built using Arduino, provides a secure and efficient approach to attendance management. Utilizing biometric technology, it accurately identifies individuals through their unique fingerprints. The Arduino microcontroller controls the fingerprint sensor, LCD display, and storage module, ensuring seamless integration. Users can easily enroll their fingerprints and mark attendance with a simple touch-based interface. This system streamlines the attendance process, enhances security, and eliminates the possibility of proxy attendance. It represents a significant advancement in attendance management, offering reliability, convenience, and tamper-proof functionality.

Project 4

Development of Compact model of InGaAs MOS transistor using TCAD augmented Machine learning.

This project focuses on creating a compact model for InGaAs Field-Effect Transistors (FETs) using Artificial Neural Networks (ANNs). The model will capture the unique electrical characteristics of InGaAs FETs, account for physical parameter variations like gate oxide thickness, and predict output characteristics under different constructional parameters and bias voltages. We will use TensorFlow for training and then implement the model in Verilog-A for compatibility with commercial circuit simulators like SPICE. This work addresses limitations in conventional compact models and offers a more adaptable and accurate solution for InGaAs FETs in semiconductor device design.
